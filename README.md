**Depends on:** **Linux** , **KDE Plasma 5**.

# Installation

```
 $ git clone https://github.com/antony-jr/VideoWall
 $ cd VideoWall
 $ cmake .
 $ make -j$(nproc)
 $ sudo make install
 $ VideoWall # Installed.
```

# Usage

**Usage is simple, you give a directory of videos or a single video file**

```
 $ VideoWall SomeVideo.mp4
```
